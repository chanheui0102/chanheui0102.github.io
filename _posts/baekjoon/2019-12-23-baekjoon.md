# Baekjoon 알고리즘 문제풀기

## [2747] 피보나치 수

문제
- 45보다 작거나 같은 자연수 N을 입력으로 받는다.

- N번째 피보나치 수를 출력한다.

정답
```c
#include <iostream>

using namespace std;

int data_dp[100];

void Fibonacci(int n)
{
	int i;

	data_dp[0] = 1;
	data_dp[1] = 1;

	for (int i = 2; i < n; i++) {
		data_dp[i] = data_dp[i - 1] + data_dp[i - 2];
	}

}
int main(void)
{
	int num;
	cin >> num;

	Fibonacci(num);

	cout << data_dp[num - 1];


	return 0;
}
```
- 매번 새로운 값을 계산하게 되어있는 기존의 코드를 작성 했을때, 시간초과 결과가 나왔다.

- 이미 계산한 값을 저장해 놓았다가 다시 불러다 사용하여 시간을 단축 시키는 DP 방식으로 다시 풀어서 정답
